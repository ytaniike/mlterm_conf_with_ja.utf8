* For mlterm ver 3.0.4

* Build & Install

Get the source files from
http://sourceforge.net/projects/mlterm/

$ ./configure --prefix=/usr --with-type-engines=xft
$ make
# make install

* Setup (the following files are located ~/.mlterm directory).

aafont: conf file for anti-alias fonts.
challenge: this file is automatically created by mlterm.
color: Original colors are too vivid for me.
font: conf file for bitmap fonts.
main: Main conf file.
